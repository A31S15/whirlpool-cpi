# ``whirlpool-cpi`` crate
Only utilizes Anchor versions 0.29.0 and above
## Usage
### with Anchor 0.29.0
Cargo.toml
```
[dependencies]
anchor-lang = "=0.29.0"
anchor-spl = "=0.29.0"
whirlpool-cpi = { git = "https://github.com/orca-so/whirlpool-cpi", branch = "anchor/0.29.0" }
```
